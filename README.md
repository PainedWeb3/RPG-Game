# RPG Arena – Turn-Based Battle Game

A simple, elegant, browser-based turn-based RPG battle simulator built with vanilla HTML, CSS, and JavaScript.

Watch randomly generated fantasy characters fight to the death (or until 10 turns pass) in a stylish arena interface.

## Features

- 5 randomly generated characters at the start of each game
- 5 unique classes with different stats, playstyles & special abilities
  - Fighter – balanced melee powerhouse
  - Paladin – tanky support with self-healing
  - Monk – fast striker with crits & small self-heals
  - Wizard – fragile glass cannon with mana-based Fireball
  - Assassin – high-riskhigh-reward crit & poison specialist
- Nice visual character cards with HPmana bars
- Turn-based combat with automatic AI opponents
- Simple AI that prioritizes low-HP targets (kill-seeking behavior)
- 40% chance to use special attack each turn
- Game ends after 10 turns or when only one fighter remains
- Clean, dark-themed interface + battle log
- Modular class-based architecture (easy to extend)

## Technologies

- HTML5
- CSS (custom dark theme + responsive cards)
- Vanilla JavaScript (ES6+ classes)
- No frameworks  build tools required

## How to Play

1. Open `index.html` in any modern browser
2. Click Start Game → 5 random heroes appear
3. Click Next Turn repeatedly to watch the battle unfold
4. The survivor with the most HP after 10 turns — or the last one standing — wins!
